Setup New Computer:     
Download and install the following programs: 
https://www.ni.com/en/support/downloads/drivers/download.frc-game-tools.html#500107 (FRC game tools)
https://packages.wpilib.workers.dev/installer/v2024.2.1/Win64/WPILib_Windows-2024.2.1.iso (WPLib for java)
https://github.com/REVrobotics/REV-Software-Binaries/releases/download/rhc-1.5.3/REV-Hardware-Client-Setup-1.5.3.exe  (revrobotics hardware configuration tool (sparkmax controller identification))
https://github.com/git-for-windows/git/releases/download/v2.43.0.windows.1/Git-2.43.0-64-bit.exe  (git is a file versioning tool)
https://download.kde.org/stable/kdiff3/kdiff3-1.10.0-windows-cl-64.exe (kdiff is a file/directory compare tool)
https://github.com/notepad-plus-plus/notepad-plus-plus/releases/download/v8.6.2/npp.8.6.2.Installer.x64.exe (Notepad++ text editor)
https://download.mythicsoft.com/flp/3420/agentransack_3420.exe  (Searching tool)
https://central.github.com/deployments/desktop/desktop/latest/win32 (Github Gui)
